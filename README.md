# Nitro & SlotBot Sniper
Snipe nitro drops and slotbot ~grabs

## Requirements
- Python 3+
- pip
  - You will need to run `pip install discord` and `pip install requests`

# Usage
- Go over to `config.json`
- Put your token where it says `PUT_TOKEN_HERE`
- Run the program
- Enjoy sniping
